Requirements

  • .NET Framework 4.6.2

Installation

  1. Extract everything from the 7zip file.
  2. Drag and drop any supported file on the AoM File Converter.exe file.
  3. Enjoy! :)

Application Information

This simple application converts anim.txt, prt, mtrl, and brg files to xml for the Age of Mythology game.

Usage

Simply drag, and drop (or use Open With) an anim.txt, prt, and/or mtrl file on the EXE to convert it to an Xml file, or the other way around to convert it accordingly.

The output file will be in the same location as the input file, but with an added extension.

Drag a Brg file on the EXE to extract all of its Mtrl files into a materials folder in the same path as the brg file.

Issues

  • If you get errors try one of the following:
    • Make sure the folder that you put the files in is not readonly.
    • Make sure you have the requirements.
    • Create an issue here.
